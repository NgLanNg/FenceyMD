# MD Reader: Tour Book

This folder is a tour of the app. **Open it in MD Reader** and
the sidebar will show ten chapters. Read them in order, or jump
to the one you need.

## The chapters

| Read | Title | Covers |

| 00 | [Welcome](00-welcome.md) | What the app is, how to navigate, where to start |
| 01 | [The Reading Experience](01-reading.md) | Typography, fonts, content width, dark mode |
| 02 | [Navigation](02-navigation.md) | Sidebar, library, recents, keyboard shortcuts |

| 09 | [Exporting to PDF](09-pdf.md) | One-click chapter PDF |

The first one to read is **00-welcome.md**. It walks you
through the rest.

## How to open this folder

1. Launch the app.
2. Click **Open Folder** in the picker, or use `Ctrl/Cmd + O`.
3. Pick this `demo/` directory.
4. The sidebar populates with the chapters. Click any title.

From there everything is the same as reading your own book.
Scroll, navigate, edit, export.

## A note on this being a real folder

There is no magic to this directory. It is plain Markdown on
disk. Rename a file, the sidebar updates. Add a `.md` file, it
appears. Delete one, it's gone. The app doesn't ship a sample
book. This *is* a real book you can give to anyone with the
app installed.
