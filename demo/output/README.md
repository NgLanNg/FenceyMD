# demo/output/

Reference renders of the demo chapters. Generated by running Chrome
headless against synthetic HTML that mirrors the in-app CSS — used to
sanity-check the PDF pipeline without rebuilding the Tauri app each
time.

- `excalidraw-demo.pdf` — sample PDF of the `06-excalidraw.md` chapter
  showing the inline-SVG conversion for Excalidraw scenes.
